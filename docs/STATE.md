# Where we are

**Branch:** intentionally not hard-coded here. Check `git branch --show-current`.

## Running the game

```sh
pnpm install
pnpm dev                 # http://localhost:5173/ (or next free port)
pnpm test                # both projects (unit + integration)
pnpm test:unit           # pure/build-time logic only, serialized for singleton safety
pnpm test:integration    # real RPG.js engine in Node via @rpgjs/testing
pnpm test:e2e:smoke      # Playwright boot smoke in real Chromium
pnpm test:coverage       # unit coverage gate, 95% lines/95% functions/90% branches/95% statements
pnpm build               # prebuild (validate + build-spine + typecheck) → vite build
pnpm preview             # serve the built bundle

GITHUB_PAGES=true pnpm build   # build with /poki-soweli/ base
CAPACITOR=true pnpm build      # build with ./ base for WebView
```

## Playtest the real game, not just the tests

Tests catch regressions in behavior the tests already asserted. Whether the game is **fun** only reveals itself in a browser. Before merging any feature that touches player-visible behavior, run `pnpm dev` and walk it yourself. Screenshots and a sentence in the PR description are better than a ticked checkbox.

## Hard rules (override defaults)

-   **Fan-tasy is the only tileset family.** `public/assets/tilesets/{core,seasons,snow,desert,fortress,indoor}/` is the source of truth.
-   **Maps are build artifacts.** Edit `scripts/map-authoring/specs/<id>.ts`, run `pnpm author:build <id>`. Never edit `src/tiled/*.tmx` or `public/assets/maps/*.tmj` directly. `pnpm author:verify` enforces `.tmx` and `.tmj` sync in `validate` + `prebuild` + CI, while `tests/build-time/map-preview-regression.test.ts` pixel-diffs all committed preview PNGs against the renderer output.
-   **Every monster is catchable.** Tiering is rarity + catch difficulty, not whether `poki` works. All 43 authored species now appear in catchable encounter tables; green dragon (`akesi_sewi`) is both the endgame set-piece and a rare final-route catch, with its dedicated defeat animation reserved for the green-dragon clear.
-   **No hand-authored toki pona.** Every user-facing TP string round-trips through the Tatoeba corpus. `pnpm validate-tp` is the gate. If it rejects, rewrite the EN.
-   **No direct `localStorage` / `IndexedDB`** in feature code — `src/platform/persistence/preferences.ts` (KV) or `src/platform/persistence/database.ts` (structured) only.
-   **Docs > tests > code.** Docs describe the game; tests describe the code; code satisfies both. Tests never chase code.
-   **Integration + E2E > unit.** Integration tests with the real engine and E2E tests with headed Chromium carry the weight. Pure-logic unit tests are for algorithms, nothing else. See `docs/TESTING.md`.
-   **Always use pull requests.** Never push to `main` directly.

-   **Maestro is scaffolded, not device-proven yet.** `pnpm maestro:check` validates the Android/iOS flow syntax locally, but the flows still need to be run on a booted Android emulator with the debug APK installed and on an iOS simulator against deployed Pages before they can close release QA.
-   **Signed release APK support is intentionally deferred.** The current `release.yml` is configured to publish a debug APK artifact for each release-please tag. A signed release APK still needs the keystore-secret path, `assembleRelease`, release-asset attachment, and physical-device QA to be implemented and proven before it becomes current-state documentation.
-   **Audio is runtime-wired, not final-scored.** The current resolver points every `BgmId` and `SfxEvent` at shipped `.ogg` assets, but some tracks and SFX events share source files. Treat distinct biome scoring, unique SFX sourcing, and broader browser-device audio QA as remaining polish, not already-finished content.
-   **The Capacitor Android scaffold is still generated, not committed.** Local native builds need `pnpm exec cap add android` when `android/` is absent.
-   **RPG.js v5 is beta.** `5.0.0-beta.1` across all `@rpgjs/*` packages. API churn is still possible; see `docs/ROADMAP.md` risk list.
-   **`@rpgjs/common/src/rooms/WorldMaps.ts`** ships with a known type error; `pnpm typecheck` filters it via pipefail grep. Revisit when v5 goes stable.
