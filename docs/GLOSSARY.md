# poki soweli — toki pona glossary

The ~120-word toki pona dictionary is the raw material. This doc maps every TP word the game actually uses to its role + approximate frequency across dialog, species names, moves, and region names. Useful for:

-   Authors: see what vocabulary is already in play before inventing a new id.
-   Playtesters: see which words a player is likely to have encountered by a given beat.
-   Translators: the rough EN sense we assume when we pick a word for a role.

Frequency is rough — `★★★` = shows up in every region, `★★` = 2-3 beats, `★` = rare.

## Core action + combat vocabulary

| TP         | EN sense                        | Role in game                                                                      | Freq |
| ---------- | ------------------------------- | --------------------------------------------------------------------------------- | ---- |
| **poki**   | container / net                 | Catch item. `poki_lili`, `poki_wawa`.                                             | ★★★  |
| **soweli** | land animal / mammal / creature | Prefix for mammal-archetype species.                                              | ★★★  |
| **waso**   | bird / winged creature          | Prefix for bird-archetype species.                                                | ★★   |
| **kala**   | fish / aquatic                  | Prefix for aquatic species.                                                       | ★★   |
| **akesi**  | reptile / non-cute animal       | Prefix for reptilian species.                                                     | ★★   |
| **jan**    | person                          | Prefix for humanoid monsters (goblin/orc/skelly) + NPC names.                     | ★★★  |
| **sijelo** | body                            | Prefix for bone/skeletal monsters.                                                | ★★   |
| **kon**    | spirit / air / breath           | Spirit-flame starter (`kon_moli`); a wawa move (`kon_wawa`).                      | ★★   |
| **pipi**   | bug / insect                    | Used by resolved corpus lines in descriptions; not a current species-name prefix. | ★    |
| **utala**  | fight / combat                  | Generic combat move; also warrior-creature name.                                  | ★★★  |
| **wawa**   | strong                          | Type (physical bruiser); name modifier for strong creatures.                      | ★★★  |

## Feedback + mastery vocabulary

Words the player learns by success/failure events:

-   **pona** (good) — positive affirmation / pass feedback
-   **ike** (bad) — negative / fail feedback
-   **ala** (nothing / none) — empty states, missing data
-   **awen** (save / stay / remain) — save-menu verb
-   **kama** (load / come / arrive) — load-menu verb
-   **tawa** (go / leave / cancel) — navigation / cancel
-   **open** (open / new / begin) — "new game"
-   **sin** (new / again) — "new" modifier
-   **moli** (dead / ended) — defeat-state word and zombie-style creature ids; avoid gore in player-facing prose
-   **pimeja** (dark / black) — mummy, dark-tone creatures
-   **pona** (good), **suwi** (sweet/cute) — positive creature descriptors
