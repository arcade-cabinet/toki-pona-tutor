# Retired Corpus Brief

The corpus/translation sourcing workflow is retired. Rivers Reckoning is authored directly in English and uses curated investigation clues in `src/content/clues.json`.

Do not add a new corpus, translation validator, or language-learning data source unless the product direction is explicitly reopened in `docs/ROADMAP.md`.
