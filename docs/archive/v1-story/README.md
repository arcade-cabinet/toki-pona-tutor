# v1 Story Archive

Archived creative docs from Rivers Reckoning v1 (finite 7-beat arc, green-dragon ending). **Not authoritative.** See `docs/DESIGN.md` for the current v2 design.

## Contents

- [STORY.md](STORY.md) — v1 story bible with region master arcs and green-dragon reveal.
- [STORY_BEATS.md](STORY_BEATS.md) — per-region authored spine.
- [LORE.md](LORE.md) — v1 world + cast + 43 species roster (species carry forward to v2).
- [JOURNEY.md](JOURNEY.md) — v1 playable-arc staging.

## Why kept

Reusable assets:
- **Species roster** (43 creatures, sprites, type tags) — unchanged in v2.
- **Tone notes** — cozy dark-fantasy, ≤ 20-word lines, no grim — preserved in v2.
- **Dialog beats** — 200-ish hand-authored lines; subset extractable into v2 dialog pool (see `docs/DIALOG_POOL.md § v1 extraction plan`).
- **Map grammar** — the 7 hand-crafted maps' biome shapes, village layouts, encounter densities informed the v2 world-gen grammar (see `docs/WORLD_GENERATION.md`).

## What's retired

- Finite story, seven beats, four region masters, green-dragon ending.
- Named cast (Selby, Rook, Marin, Frost, Cliff, etc.) as persistent characters — becomes role-archetype seed material.
- Quest chains, badge gating, proofs, post-clear loop.
- All `badge_*`, `proofs_all_four`, `green_dragon_defeated`, `game_cleared` flags.
